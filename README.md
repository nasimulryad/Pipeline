# Pipeline

A free, open-source job search tracker with optional AI-powered application tools.

## What it does

Pipeline helps job seekers stay organized throughout their search. Track every job you're considering or have applied to, store your application documents, and monitor your progress through each stage of the hiring process.

For users who connect their own Anthropic API key, Pipeline also generates tailored resumes and cover letters for each job automatically.

## Features

**Free — no account required to explore, no cost ever**
- Job tracker dashboard with pipeline/timeline view
- Color-coded stages: Considering → Applied → In Progress → Offer → Rejected
- Store job posting details, recruiter contacts, and notes per job
- Upload and organize documents per application (resume, cover letter, etc.)
- Active jobs and potential jobs at a glance

**With your own Claude API key (bring your own — we never store it)**
- AI-generated tailored resume per job posting
- AI-generated cover letter per job posting
- Job posting analysis — key requirements, fit assessment, red flags
